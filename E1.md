# Entregable 1: Tabla Detallada de Casos de Prueba

## Sistema ERP Integral - Módulos: Ventas, Marketing, RRHH y Postventa

---

## 1. Módulos a Probar

| Módulo                  | Descripción                                           |
| ----------------------- | ----------------------------------------------------- |
| **Marketing**           | Gestión de Leads, Calificación y Conversión a Cliente |
| **Ventas**              | Cotizaciones, Facturación, Impuestos y Oportunidades  |
| **RRHH**                | Nómina, Horas Extra, Empleados                        |
| **Postventa (Soporte)** | Tickets de Soporte, Prioridades                       |

---

## 2. Integraciones Clave

- **Marketing → Ventas**: Traspaso de Lead calificado a Cliente y Oportunidad
- **Ventas → Postventa**: Creación de ticket de soporte asociado a producto comprado
- **Seguridad**: Restricción de acceso por rol (Marketing no puede ver RRHH)

---

---

## 5. Criterios de Prueba

### Criterios de Entrada

- Código desplegado en entorno de QA
- Base de datos configurada con datos de prueba
- Historias de Usuario (HU) documentadas
- smoke tests passed

### Criterios de Salida

- 100% de las pruebas críticas de Ventas y RRHH aprobadas
- cero defectos bloqueantes o críticos activos
- 95% de los casos de prueba superados
